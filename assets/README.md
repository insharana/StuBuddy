# Study Buddy Matcher

## Overview
Study Buddy Matcher is a university-focused platform to help students find study partners based on subject, semester, and availability. It ensures privacy and organized communication.

## Features
- University email login
- Automatic department & semester placement
- Subject selection
- Post study problems
- Offer Help for private messaging
- Events & deadlines section

## Technologies
- HTML, CSS, JavaScript
- Firebase (for authentication & database)

## How to Run
1. Clone the repository
2. Open index.html in a browser
3. Follow instructions on the page to test features
